# Board Overview

### Board info

- Manufacturer: GIGABYTE
- Model: H410M S2H V2, revision 1.7 (per provided URL; there is also rev 1.0, but this guide targets rev 1.7 specifically).
- Form factor: Micro‑ATX.
- Socket: LGA1200.
- Chipset: Intel H470/H410 series (marketed as H410M; functionally a 400‑series LGA1200 desktop board).
- Memory: 2× DDR4 DIMM slots, up to 64 GB, typical speeds up to DDR4‑2933 depending on CPU.
- Expansion: 1× PCIe 3.0 x16, 2× PCIe 3.0 x1.
- Storage: 4× SATA 6 Gb/s ports, 1× M.2 (PCIe 3.0 x4/x2 and SATA support).
- Rear I/O highlights: D‑Sub, DVI‑D, HDMI, 2× USB 3.2 Gen1, 4× USB 2.0, 1× RJ‑45, 3× audio jacks.

Onboard controllers (rev 1.7, based on vendor specs plus typical community confirmation for this model family):

- LAN: Intel Gigabit Ethernet (Intel GbE LAN, behaves like I219‑V class; treat as IntelMausi‑compatible).
- Audio: Realtek HD Audio codec (Realtek ALC8xx‑class, usually ALC887/892; we will treat as AppleALC‑compatible and note the exact codec as an assumption).
- WiFi/BT: None onboard; any WiFi/BT would be via PCIe/USB add‑on card, which you must choose for macOS compatibility.

### CPU family \& CPU lines

The board supports Intel 10th‑gen Comet Lake desktop CPUs on LGA1200:

CPU FAMILY PROFILE:

- Intel Comet Lake‑S (10th‑gen Core, Pentium Gold, Celeron) desktop.

From Gigabyte’s CPU support and typical 400‑series behavior, realistic CPU lines for this board:

1. **Comet Lake iGPU line (UHD 630, i3/i5/i7/i9 with graphics)**
   - Example CPUs: i3‑10100, i5‑10400, i5‑10500, i7‑10700, i9‑10900.
   - Integrated GPU: Intel UHD Graphics 630 (Gen9.5).
2. **Comet Lake budget iGPU line (Pentium/Celeron with UHD Graphics)**
   - Example CPUs: Pentium G6400, Pentium G6500, Celeron G5900.
   - Integrated GPU: Intel UHD Graphics (still Gen9.5, similar handling to UHD 630 but with fewer EUs).
3. **Comet Lake “F” / no‑iGPU line**
   - Example CPUs: i5‑10400F, i7‑10700F, i9‑10900F.
   - No functional iGPU; requires a supported dGPU.

On this platform, the iGPU lines (1 and 2) are the primary and recommended paths for Hackintosh due to simpler setup and good macOS compatibility.
